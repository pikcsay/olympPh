Задание проекта OlymPhysics

В программе будет четыре виджета:
1)Виджет олимпиады производит доступ к базе данных с датами туров и сайтами олимпиад.
2)Виджет сайты позволяет добавлять ссылки на полезные ресурсы в базу данных(кнопка 
'добавить сайт') и удалять ссылки. Сайты можно будет запустить прямо в программе.
3)Виджет заметки позволяет пользователю добавлять мероприятия с датами в бд и получать
список дел в порядке их совершения. Также мероприятия будут подчищаться по мере их
прохождения.
4)Вкладка задачи позволяет полюзователю пользоваться бд с задачами: 
добавлять задачи(кнопка 'добавить задачу') и получить случайную задачу(кнопка 'случайная
задача'). Также будет кнопка 'удалить задачу' и кнопка 'получить задачу', которая 
производит поиск с фильтром.

Структура:
- OlympPhysics: главное окно программы
	createGroupSites - создание группы кнопок сайтов
	filter - поиск олимпиад по турам
	searchOlymp - поиск олимпиад по названию
	saveNotes - сохранение заметок
	showNotes - отображение заметок
	delNotes - удаление заметок
	addSite - добавление сайта
	delSite - удаление сайта
	ViewWebSite - отображение страницы сайта
	addProblem - добавление задачи
	getRandProblem - получение случайной задачи
	getProblems - получение задач по темам
- WidSites: класс виджета для сбора кнопок с сайтами
	addWidget - добавление виджета сайта
	init - заполнение лайаута кнопками
- WidProblems: класс виджета для сбора лэйблов с картинками задач
	init - заполнение лайаута лэйблами
- ViewProblems: доп. окно программы для отображения задач по темам
	initUI - заполнение окна
